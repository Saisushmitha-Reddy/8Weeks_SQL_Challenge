# Foodie-Fi — SQL Case Study (MySQL)

Foodie-Fi is a subscription-based streaming service case study.
This analysis focuses on subscription behavior, churn, upgrades,
and revenue trends.

## Files
- `schema.sql` — database schema and seed data
- `solutions.sql` — SQL answers to all case study questions

## Key Skills Demonstrated
- Subscription lifecycle analysis
- Cohort analysis
- Date-based filtering
- Churn rate calculation
- Window functions
- Revenue analysis
